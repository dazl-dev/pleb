# Pusha

Custom publishing cli for lerna/yarn/workspces project

## Installation

Install `pusha` as a dev dependency:

```
yarn add pusha --dev
```

## Usage

```
pusha "path/to/your/repo"
```

## CLI Options

```
  -v, --version                       output the version number
  -h, --help                          output usage information
```

# Lerna Publisher

Custom publishing cli for lerna/yarn/workspces project

## Installation

Install `lerna` :

```
nom install lerna-publisher -g
```

## Usage

```
lerna-publisher "path/to/your/repo"
```

## CLI Options

```
  -v, --version                       output the version number
  -h, --help                          output usage information
```
